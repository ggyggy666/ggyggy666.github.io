PHP连接MySQL数据库操作--面向过程

PHP学习 PHP连接MySQL数据库操作--面向过程 2020/05/19

今天继续学习PHP连接数据库–面向过程。

连接数据库

<?php
    $conn = mysqli_connect($servername, $username, $password, $dbname");
    if(mysqli_connect_error($conn)){
        echo "连接失败：".mysqli_connect_error();
    }
    mysqli_close($conn);
?>

创建数据库

在没有数据库之前，连接数据库就不用加上dbname，然后先连接上后再创建数据库。

$conn = mysqli_connect($servername, $username, $password);
...
$sql = "create database ggy";
if(mysqli_query($conn, $sql)){...}    //执行SQL语句
else echo "error: ".mysqli_error();    //输出错误

选择使用某数据库

$conn = ...
创建一个数据库后
mysqli_select_db($conn, "ggy"); //相当于use ggy

创建表、插入、删除、更新

其实也就是SQL语句变了而已。只要记住：

1. mysqli_select_db($conn,"ggy") //先选择数据库
2. mysqli_query($conn, $sql)     //再执行SQL语句
3. mysqli_error()                //出错就报错

查询数据库

跟面向对象一样，查询到的数据是按行记录的，然后每一行都可以用一个函数将其构造成一个关联数组，然后while循环就行了。

$result = mysqli_query($conn, "select * from ggy");
while($row = mysqli_fetch_array($result)){
    echo $row['username'].$row['password'];
}

之后，条件查询和order by也就是SQL语句变换一下而已。
