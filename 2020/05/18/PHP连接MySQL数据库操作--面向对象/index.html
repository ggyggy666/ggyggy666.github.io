PHP连接MySQL数据库操作--面向对象

PHP学习 PHP连接MySQL数据库操作--面向对象 2020/05/18

本文于 407 天之前发表，文中内容可能已经过时。

好久没更新了，最近忙于学习PHP。学会了如何连接数据库，在此总结一下。

连接MySQL数据库有面向对象和面向过程两种。本篇是面向对象。

创建连接

<?php
    $servername = "localhost";    //数据库所在地址，本地搭建的因此是localhost或者127.0.0.1
    $username = "username";        //数据库用户名
    $password = "password";        //数据库密码

    $conn = new mysqli($servername, $username, $password);        //创建连接
    if($conn->connect_error){
        die("连接失败：".$conn->connect_error);        //die终止连接并抛出错误信息
    }
    echo "连接成功";
    $conn->close();                //手动关闭连接
?>

创建数据库

接上面的连接数据库后，可以创建一个数据库。

$sql = "create database ggy";
if($conn->query($sql) === true){
    echo "数据库创建成功";
}else{
    echo "Error creating database: ".$conn->error;
}

$conn->query()是执行里面的SQL语句并返回true或者false。

创建数据表

要创建表，首先得指定是在哪个数据库之下，即：use ggy。表明使用ggy这个数据库。面向对象的话在创建连接的时候就可以传数据库名称进去（前提是知道这个数据库名称). 如下：

$dbname = "ggy";
$conn = new mysqli($servername, $username, $password, $dbname);
...
$sql = "create table ggy_table(
id int(6) unsigned auto_increment primary key,
username varchar(30) not null,
password varchar(30) not null)";

$conn->query($sql);

注意，创建表时里面的字段要指定类型以及长度。auto_increment是当新增一条记录时，id能自动加1. 还要设定主键，其它字段如果是必需的就得使用not null。

插入数据

$sql = "insert into ggy_table(username, password) values ('ggy', 666)";
$conn->query($sql);

看到这里就知道了，其实在连接上数据库后，就是使用SQL语句，然后使用$conn->query($sql)执行就可以了。

插入多条数据

其实就是使用了堆叠查询的原理。

$sql = "insert into ggy_table(username, password) values ('ggy','ggy');";    //SQL语句后有分号
$sql .= "insert into ggy_table(username, password) values ('ggg', 'handsome')";
$conn->query($sql);

使用分号隔离每个SQL语句，使用.连接起来，然后再一次执行即可。这样的话需要写很多行，而且要写很多重复的代码，也可以这样写：

$sql = "insert into ggy_table(username, password) values ('ggy','666'),('fff','admin')";

只需要将插入的数据用括号括起来，相互之间用逗号分隔即可。

读取数据

使用select查询出数据，然后使用fetch_assoc()将这些数据按每一行记录来作为一个关联数组，然后就可以通过数组名键来访问值了。

$sql = "select id, username, password from ggy_table";
$result = $conn->query($sql);    //查询到的多条数据赋值给result
if($result->num_rows > 0){        //因为记录是按行的，因此通过判断它的行数来判断查到的数据是否为空。
    while($row = $result->fetch_assoc()){
        echo $row["id"].$row["username"].$row["password"];
    }
}

使用了一个while循环，会使每次调用fetch_assoc()之后相当于遍历每一行查到的数据，然后将该行记录变为一个关联数组。

where条件、order by排序查询

$sql = "select * from ggy_table where username='ggy'";
$result = $conn->query($sql);
...

与上面select的一样，排序的话就是在SQL语句后面直接加order by xx ，默认是升序，要降序的话，后面加DESC。

更新数据

$sql = "update ggy_table set username='ggy666' where username='ggy'";
$conn->query($sql);

将用户名为ggy的所有记录的用户名都更改为ggy666.

删除数据

使用delete删除的是表中的某些记录。

$sql = "delete from ggy_table where username='ggy'";
$conn->query($sql);

删除ggy_table中所有用户名为ggy的记录。

预处理语句

使用预处理语句可以有效防止SQL注入。因为预处理语句对用户的输入都不会当做SQL语句执行，而是直接输出。

//sql连接后
$stmt = $conn->prepare("insert into ggy_table(username, password) values (?,?);
$stmt->bind_param("ss", $username, $password);
//上面只是预备好插入的SQL语句，值都是？。然后绑定就是将之后传入的$username, $password代替？
从而组成完整的SQL语句。ss是代表类型。每个参数类型都是字符型，用s表示，两个参数，因此是ss。

$username = 'ggy';
$password = '666';
$stmt->execute();    //将参数值代替？，然后执行SQL语句。

$username = 'ggg';
$password = '666';
$stmt->execute();    

$stmt->close();
$conn->close();

由于已经设定了预处理模板，那么之后就可以任意的设定参数值然后传进去执行。
