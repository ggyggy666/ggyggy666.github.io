Js学习（四）—— 类、错误处理、DOM

前端学习 Js学习（四） 2020/05/03

本文于 401 天之前发表，文中内容可能已经过时。

今天继续学习JS。

Math类

JS中也需要用到一些数学方法，通过使用Math类即可。

  1. 向上取整：Math.ceil(x)
  2. 向下取整：Math.floor(x)
  3. 随机数：Math.random()
  4. 平方根：Math.sqrt(x)
  5. 四舍五入：Math.round(x)
  6. 求最值：Math.max(a,b,c…)和Math.min(a,b,c…),参数不限。

Date类

首先定义一个时间：var x=new Date()；

  1. x.getFullYear(), 返回x对应的年份。
  2. x.setFullYear(year, month, day)，设置年份，month和day可选。
  3. x.getMonth(), 返回x的月份
  4. x.setMonth(month, day), 设置月份，day可选。
  5. x.getDate()，返回x在一个月中的第几天。
  6. x.getDay(), 返回x在一周的第几天。
  7. x.setDate(day), 设置一个月的第几天。
  8. x.toString()，将日期转为字符串。

捕获和处理错误

JS的捕获错误其实跟其他语言类似。也是用try-catch,throw。如：

try{
    ...
}catch(err){
    console.log(err.message);    //通过使用err.message返回错误信息。
}

但是这个错误信息是原本就被定义了的，如果使用自定义的错误，那么需要new Error(“错误信息…”)，而且是使用throw抛出的。如：

try{
    ...
    throw new Error("错误");
}catch(err){
    console.log(err.message);    //通过使用err.message返回自定义的错误信息。
}

DOM

DOM是一个文档对象模型，组成文档的对象通过树形结构连接在一起，表示层次关系。文档元素就是一个标签以及之间的内容，对应一个对象。所以节点树就是HTML的元素的结构。根节点是html标签，然后是head、body标签…如果要使用js去对文档元素进行操作，首先就得获取到这个文档元素。

获取文档元素

  1. 通过id。document.getElementById(“”). innerText是开始标签与闭合标签之间的内容。
  2. 通过类名。var s = document.getElementsByClassName(“”). 注意同一个类名的标签不唯一。因此返回的是一个数组，需要使用s[0],s[1]…获取到对应的元素。
  3. 通过标签名。var s = docuemnt.getElementsByTagName(“div”). 同样返回的是数组。
  4. 通过document.querySelector(“.类名”), 这个需要满足有类名，而且这个类有css样式，所以才会是.类名。而且只返回第一个元素。需要用querySelectorAll()返回所有匹配的元素。

节点树的操作

首先获取一个元素节点：var s = document.getElementById(“”);

  1. 获取第一个子节点。var s1 = s.firstElementChild;
  2. 获取最后一个子节点：var s2 = s.lastElementChild;
  3. 获取前一个兄弟节点：var s3 = s2.previousElementSibling; 当然第一个节点没有前一个兄弟节点。
  4. 获取后一个兄弟节点：var s4 = s1.nextElementSibling; 当然最后一个节点没有兄弟节点。
  5. 子节点列表：children,是一个数组，需要使用：s.children[0]…获取它的长度：s.children.length;

获取和设置属性

有两种方法。

  1. 通过使用: s.属性名，返回的是这个属性对应的值（数值或者字符串等）。可以用s.属性名=xxx，直接赋值。
  2. 使用getAttribute(), 如：s.getAttribute(“属性名”). 返回的是字符串。相反，可以用s.setAttribute(“属性名”,”属性值”).

需要注意，如果属性名是class等，不能直接用s.class, 用s.className, 而可以直接使用：s.getAttribute(“class”)

节点的操作

建立在父节点与子节点的基础上。

  1. 创建节点：document.createElement(“a”),a是超链接标签名。
  2. 插入节点：一：插入到最后一个子节点中。node1.appendChild(node)。二：插入到某个节点前面。父节点.insertBefore(node1,node2)，将node1插进父节点的子节点中，并且排在node2前面。
  3. 删除节点。父节点.removeChild(子节点)。
  4. 替换节点。父节点.replaceChild(a,b)。将a替换掉b节点。
